# Matheesha Gamage Portfolio

This is a modern, animated, and professional personal portfolio website built with React and Tailwind CSS.

## 🚀 How to Run

1. Install Node.js and npm
2. Run:
   npm install
   npm start

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Lucide React Icons
